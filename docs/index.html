都道府県の宿泊者数の重心

当サイトについて

当サイト「都道府県の宿泊者数の重心」は観光庁「宿泊旅行統計調査」に収められた「施設所在地、居住地別延べ宿泊者数（従業者数100人以上の施設）」のデータを可視化するツールです。

ある都道府県に訪れた宿泊客がどの方面から来たのか、またその動向がどう変わってきたかについて知ることができます。詳しくは「使い方について」をご覧ください。

宿泊者数の重心とは？

ある都道府県を訪れた宿泊客の居住地別の延べ数を使って47都道府県の地理的中心（mean center）を求めたものです。

宿泊者一人ひとりに等しい重みを与えて、それぞれの居住地の都道府県庁に宿泊者を集めたとき、47の都道府県をちょうどバランスよく支えることのできる場所が「都道府県の宿泊者数の重心」です。

詳しくは「算出方法について」をご覧ください。
